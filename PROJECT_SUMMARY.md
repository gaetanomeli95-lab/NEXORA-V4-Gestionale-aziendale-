## 🚀 **Ready to Use**

### ⚡ **5-Minute Setup**
```bash
cd "c:\Users\User\Desktop\NEXORA v4"
npm install --legacy-peer-deps
copy .env.example .env.local
npx prisma generate
npx prisma db push
```
